Lottery

The lottery can't be run without any eligible artists.

This section will allow you to run the lottery.

  • Running the lottery will erase the current lottery and standby lists.
  • The results are picked from artists that are marked as 'eligible' for the lottery.
  • Artists checked below will be considered before anyone else.
  • Artists that are picked will be added to the top of the standby list.
  • You may manually check (or uncheck) guaranteed winners for this lottery below.
